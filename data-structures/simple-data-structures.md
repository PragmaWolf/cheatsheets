# Простые структуры данных - типы данных

Примитивы, базовые элементы. Составная часть сложных структур данных.

> [!NOTE]
> Могут быть дополнительные, в зависимости от языка программирования.

- Целые (integer).
- С плавающей точкой (float).
- Булевые (boolean).
- Строчные (character).

**Операции:**
- Создание: да
- Вставка: нет
- Обход: нет
- Поиск: нет
- Сортировка: нет
- Слияние: нет
- Обновление: да
- Удаление: да
